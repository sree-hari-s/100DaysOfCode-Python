# Workout Tracking

Utilizing the [nutritionix](https://www.nutritionix.com/business/api) api and [sheety](https://sheety.co/) to interact with the Google sheets created a workout tracking cli tool that calculates the duration and calories used in a workout.

## Concepts used in this mini project

- Python datetime strftime()
- APIs and making post requests
- Authorization and Headers
- Environment variables usage
